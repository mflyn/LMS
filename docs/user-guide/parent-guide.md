# 家长操作指南

家长端用于管理家庭与孩子，安排五育任务，记录成长过程，并通过周报、星星和
奖励提供反馈。

## 1. 登录和家庭初始化

### 注册与登录

- 新用户打开“注册家长账号”，填写用户名、称呼、邮箱和密码后选择“注册”。
- 已有账号打开“家长登录”，输入用户名和密码后选择“登录”。
- 账号不能进入家长端时，页面会提示“该账号不能进入家长端”。
- 使用完毕后选择页面右上角的“退出登录”。

家长会话失效时，系统会清除当前登录状态并返回登录流程。重新登录后可继续
操作。

### 创建家庭

首次进入时，在“创建家庭”页面填写家庭名称并选择“创建家庭”。当前 Web
页面只需填写家庭名称，家庭日期按 `Asia/Shanghai` 时区处理。

## 2. 家庭和孩子管理

打开“孩子”可以查看 Family ID、已有孩子及每个孩子的 Child ID。

### 添加孩子

1. 选择“添加孩子”。
2. 填写必填的“孩子姓名”。
3. 按需填写昵称、年级和学校。
4. 保存孩子档案。

### 编辑孩子档案

1. 在目标孩子条目中选择“编辑档案”。
2. 修改姓名、年级、学校和教材版本。
3. 按需填写兴趣、薄弱学科、体育偏好、艺术兴趣、劳动习惯和品德目标；多个项目使用中文或英文逗号分隔。
4. 可上传、替换或移除头像，然后选择“保存孩子档案”。

头像上传完成后仍需保存档案才会生效。上传过程中不能关闭或保存；上传完成后若直接
关闭编辑窗口，新上传的草稿头像会被清理，原头像保持不变。

### 设置或重设 PIN

1. 在目标孩子条目中选择“设置 PIN”。
2. 输入 4 至 6 位数字。
3. 保存后，将新 PIN 私下告诉孩子。

重设 PIN 会使孩子原有会话失效。孩子需要用新 PIN 重新登录。

### 切换当前孩子

家长端各业务页顶部有“当前孩子”选择器。切换后，任务、记录、知识与能力点、错题、周报、
提醒和奖励都会按新选择的孩子重新加载。操作前应先确认当前孩子正确。

## 3. 家长端导航

| 导航 | 用途 |
| --- | --- |
| 今日 | 查看当天任务、五育概览和提醒 |
| 任务 | 创建、筛选、完成、确认、取消或归档任务 |
| 记录 | 添加和编辑成长记录 |
| 知识与能力点 | 按五育维度维护知识点、能力点或习惯点 |
| 错题 | 记录智育错题并持续复盘 |
| 周报 | 查看每周汇总并填写家长反馈 |
| 提醒 | 查看提醒并设置提醒类型和免打扰时段 |
| 星星与奖励 | 查看星星、创建奖励并执行兑换 |
| 孩子 | 添加孩子、编辑完整档案和头像，并设置 PIN |

在窄屏页面中，先选择“导航”打开菜单。

## 4. 任务管理

### 创建任务

1. 打开“任务”，选择“新建任务”。
2. 选择成长维度：德育、智育、体育、美育或劳育。
3. 填写任务标题、活动领域、截止日期和任务类型。
4. 智育任务还需要选择学科。
5. 按需填写预计用时、目标数量、单位、优先级和说明。
6. 可以添加多张任务附件，然后保存。

任务类型包括练习、锻炼、习惯、阅读和劳动。页面可按时间范围、成长维度和
任务状态筛选。

### 状态和操作

| 状态 | 含义 | 家长可执行的主要操作 |
| --- | --- | --- |
| 待完成 | 孩子尚未提交完成情况 | 编辑、完成、取消 |
| 待家长确认 | 孩子或家长已提交完成记录 | 确认并发放星星 |
| 已确认 | 家长已确认完成 | 查看、归档 |
| 已取消 | 任务不再执行 | 查看 |
| 已归档 | 任务退出当前工作列表 | 查看历史 |

家长代为完成任务时，可填写实际用时、实际完成数量、完成难度、是否需要帮助和
孩子备注。确认时选择“确认并发放星星”。如果确认因临时错误失败，先重新加载；
任务仍显示“待家长确认”时可以重试。如果任务已显示“已确认”但星星没有更新，
不要归档，交由系统维护人员处理。

## 5. 成长记录

“记录”适合补充没有从任务发起的活动，例如一次家庭劳动、户外锻炼或艺术
参观。

1. 打开“记录”，选择“记录成长”。
2. 选择日期和成长维度。
3. 填写领域；智育记录可以填写学科。
4. 按需填写时长、完成数量、单位、专注程度、难度、身体状态和心情。
5. 填写必填的“记录内容”，并可添加孩子自评和家长备注。
6. 选择“保存成长记录”。

已有记录可通过“编辑”更新。筛选时可选择日期范围和成长维度。

## 6. 知识与能力点

“知识与能力点”用于手动维护孩子在五育维度中的持续关注项。智育使用“学科”，
德育、体育、美育和劳育使用“领域”。

1. 先确认顶部“当前孩子”。
2. 选择“新增知识或能力点”，选择成长维度。
3. 智育填写学科，其他维度填写领域；再填写名称和初始掌握程度。
4. 保存后可按维度、学科、领域和掌握程度筛选。
5. 选择条目旁的“更新”，维护掌握程度、练习次数、错误次数和最后复习时间。

掌握程度包括未开始、学习中、基本掌握、熟练和需要复习。同一孩子下重复的维度、
学科或领域及名称组合不能重复创建。

## 7. 错题复盘和私有附件

错题仅用于智育学习复盘。打开“错题”后可以按学科和复习状态筛选。

### 记录错题

1. 选择“记录错题”。
2. 填写学科，可按需填写知识点。
3. 选择错误原因：概念不清、计算错误、粗心、方法不当或其他。
4. 按需设置提醒复习日期，填写正确答案、孩子解释和家长备注。
5. 可分别添加题目附件和答案附件；每组最多 10 个，可混合使用图片和 PDF。
6. 选择“保存错题”。

通过“复盘”可以更新孩子解释、家长备注以及“已订正、已复习、已掌握”状态，
然后选择“保存复盘”。已有错题的学科不能在复盘时修改。

### 附件规则

- 图片支持 JPEG、PNG 和 WebP；任务、错题题目和错题答案还支持 PDF。
- 每个文件不超过 10 MiB，PDF 最多 50 页；加密、损坏或包含主动内容的 PDF 会被拒绝。
- 孩子头像只支持上述图片格式，不支持 PDF，并作为私有媒体保存。
- 任务附件最多 100 个；每道错题的题目附件和答案附件各最多 10 个。
- 图片通过明确的预览操作打开；PDF 通过明确的下载操作保存，不在页面内嵌执行。
- 删除附件后先保存任务或错题，系统才会解除业务引用；取消未保存表单会清理本次新上传的草稿。

不要把家庭标识、孩子 PIN 或其他敏感信息放进附件。

## 8. 周报

1. 打开“周报”。
2. 在“周一开始”选择要查看的一周。
3. 查看记录天数、成长投入、任务完成率、新增错题、五育投入、建议复习和系统
   建议。
4. 填写“家长寄语”或“下周建议”。
5. 选择“保存周报反馈”。

当前周的统计会随本周数据变化。已经结束的一周会保留当时的统计结果，家长反馈
仍可继续更新。

## 9. 提醒设置

“提醒”显示当天提醒，并可配置以下项目：

- 任务到期提醒
- 任务逾期提醒
- 错题复习提醒
- 五育均衡提醒
- 周报提醒及周报提醒日
- 免打扰开始和免打扰结束

修改后选择“保存提醒设置”。如果页面显示“部分数据暂不可用”，其余已显示的
信息仍可查看；不要把暂不可用的数据理解为零，稍后重新加载即可。

## 10. 星星与奖励

“星星与奖励”显示当前星星、家庭奖励和星星记录。

### 创建奖励

1. 选择“新建奖励”。
2. 填写“奖励名称”和“所需星星”。
3. 选择“保存奖励”。

### 兑换奖励

1. 在可用奖励上选择“兑换”。
2. 核对奖励名称和所需星星。
3. 选择“确认兑换”。

星星不足时不能完成兑换。临时失败时保留当前结果，选择“重试兑换”。只有家长
可以执行兑换，孩子端只展示奖励和星星情况。

## 11. 常见问题和处理

### 页面一直显示正在加载

等待当前请求完成。出现“暂时无法加载数据，请重试”时，选择“重新加载数据”；
如果多次失败，确认网络和服务状态后再试。

### 页面显示部分数据暂不可用

页面会继续显示成功加载的内容，并列出暂不可用的数据来源。不要按零数据作判断，
稍后重新加载缺失部分。

### 保存时提示字段错误

检查必填项、日期和数值范围。智育任务需要学科；附件必须符合类型、大小、页数和数量限制。

### 任务状态已经变化

其他页面或家庭成员可能已经处理该任务。重新加载任务，按最新状态继续操作，
不要重复提交。

### 孩子无法登录

在“孩子”页面核对 Family ID 和 Child ID，并重新设置 4 至 6 位 PIN。连续多次
失败时，让孩子稍后再试。重设后孩子必须使用新 PIN。

### 奖励无法兑换

先核对当前星星是否达到“所需星星”。余额足够但仍临时失败时，选择“重试兑换”，
不要连续创建重复奖励。

## 12. 家庭隐私建议

- 不在公共渠道分享 Family ID、Child ID、PIN、家长密码或含这些信息的截图。
- 家长和孩子使用各自入口，不要让孩子使用家长账号代替 PIN 登录。
- 私有附件只从对应业务页面查看或下载，不要把内容或短时访问地址转发到公共位置。
- 在共享设备上使用完毕后，家长选择“退出登录”，孩子选择“退出孩子端”。
