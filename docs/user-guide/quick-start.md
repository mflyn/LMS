# 5 分钟快速上手

本指南用一个体育任务串联家长端和孩子端。完成后，家庭应能看到任务从
“待完成”进入“等待家长确认”，再由家长确认并发放星星。

## 1. 注册家长账号

1. 打开“注册家长账号”。
2. 填写用户名、称呼、邮箱、密码和确认密码。
3. 用户名至少 3 个字符，密码至少 8 个字符，两次密码必须一致。
4. 选择“注册”。

注册成功后会进入家长端。如果已经有账号，从“家长登录”使用用户名和密码
登录即可。

## 2. 创建家庭

1. 在“创建家庭”页面填写家庭名称。
2. 选择“创建家庭”。

当前家庭按 `Asia/Shanghai` 时区处理日期。创建完成后，系统会进入家庭成长
跟踪页面。

## 3. 添加孩子并设置 PIN

1. 打开家长端“孩子”。
2. 选择“添加孩子”。
3. 填写“孩子姓名”；昵称、年级和学校可以按需填写。
4. 保存后，在孩子条目中选择“设置 PIN”。
5. 输入 4 至 6 位数字并保存。
6. 记下页面显示的 Family ID 和该孩子的 Child ID。

将 Family ID、Child ID 和 PIN 私下告诉孩子。不要把这些信息放进公共截图。

## 4. 创建一个体育任务

1. 确认页面顶部“当前孩子”选择的是目标孩子。
2. 打开“任务”，选择“新建任务”。
3. “成长维度”选择“体育”。
4. 填写任务标题，例如“跳绳训练”。
5. 填写活动领域、截止日期和任务类型。
6. 可以填写预计用时、目标数量、单位、优先级、说明和任务附件。
7. 保存任务。

只有“智育”任务需要填写学科。任务附件支持多张 JPEG、PNG 或 WebP 图片，
每张不超过 10 MiB。

## 5. 孩子登录并完成任务

1. 退出家长端，或在另一个浏览器窗口打开“孩子登录”。
2. 输入 Family ID、Child ID 和 PIN。
3. 选择“进入我的成长空间”。
4. 在“今天”的“今天的任务”中打开刚创建的任务。
5. 填写实际用时、实际完成数量、完成难度和“我的一句话”。
6. 如需家长帮助，勾选“我需要帮助”。
7. 选择“提交完成情况”。

提交成功后会显示“已提交，等待家长确认。”孩子不能自行发放星星。

## 6. 家长确认并发放星星

1. 使用家长账号重新登录。
2. 打开“任务”，找到状态为“待家长确认”的任务。
3. 检查孩子提交的完成情况。
4. 选择“确认”，再选择“确认并发放星星”。

如果确认操作因临时错误失败，请重新加载任务。仍显示“待家长确认”时可以重试；
如果任务已显示“已确认”但星星没有更新，不要归档，交由系统维护人员处理。

## 7. 查看结果

- 打开“星星与奖励”，查看当前星星和星星记录。
- 打开“周报”，在“周一开始”中选择本周，查看任务完成率和五育投入。
- 需要给孩子留言时，填写“家长寄语”或“下周建议”，选择“保存周报反馈”。

至此，首次家庭成长闭环已经完成。后续可以继续创建德育、智育、美育和劳育
任务，并在“记录”中补充不依赖任务的成长活动。
