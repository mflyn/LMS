# 家庭成长跟踪用户指南

家庭成长跟踪面向家长和孩子，记录德育、智育、体育、美育、劳育五个维度的
成长任务和完成情况。家长负责家庭、孩子、任务和反馈管理，孩子通过独立入口
查看并完成自己的任务。

## 指南入口

| 指南 | 适用对象 | 内容 |
| --- | --- | --- |
| [快速上手](./quick-start.md) | 首次使用的家庭 | 从家长注册到孩子完成任务、家长确认的完整流程 |
| [家长操作指南](./parent-guide.md) | 家长 | 家庭与孩子、任务、记录、错题、周报、提醒和奖励 |
| [孩子操作指南](./child-guide.md) | 孩子及陪同操作的家长 | PIN 登录、今日任务、错题复习、成就和退出 |

## 使用入口

| 角色 | 入口 | 登录方式 |
| --- | --- | --- |
| 家长 | “注册家长账号”或“家长登录” | 用户名和密码 |
| 孩子 | “孩子登录” | Family ID、Child ID 和 4 至 6 位 PIN |

首次使用时，家长需要先完成注册、创建家庭、添加孩子并设置 PIN。Family ID、
Child ID 可以在家长端“孩子”页面查看。

## 当前功能边界

- 家长端导航为“今日、任务、记录、错题、周报、提醒、星星与奖励、孩子”。
- 孩子端导航为“今天、错题、成就、我的”。
- 孩子只能查看自己的数据，不能创建任务、修改奖励或执行兑换。
- 错题用于智育复盘；德育、体育、美育和劳育活动通过任务与成长记录跟踪。
- 当前版本不包含教师、班级、成绩、邮箱找回密码或原生移动应用。

## 隐私提示

- Family ID、Child ID 和孩子 PIN 只在家庭成员之间私下传递，不要发布到群聊、
  公共文档或截图中。
- 家长重设 PIN 后，孩子需要使用新 PIN 重新登录。
- 任务附件和错题图片属于私有内容，应从对应页面的“查看”操作打开。
- 不要在用户指南、反馈或截图中粘贴账号密码、PIN 或服务配置。

文档内容以当前 Web 界面为准。部署地址由实际运行环境决定，因此指南不固定
具体域名。
