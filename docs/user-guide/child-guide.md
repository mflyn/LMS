# 孩子操作指南

孩子端只显示当前孩子自己的任务、待复习错题、星星、奖励和个人资料。请先向
家长取得 Family ID、Child ID 和 PIN。

## 1. 使用 PIN 登录

1. 打开“孩子登录”。
2. 输入家长提供的 Family ID。
3. 输入自己的 Child ID。
4. 输入 4 至 6 位 PIN。
5. 选择“进入我的成长空间”。

出现“家庭、孩子或 PIN 不正确”时，先检查输入内容，再请家长到“孩子”页面
核对标识或重新设置 PIN。出现“尝试次数过多，请稍后再试”时，等待一段时间后
再登录。

PIN 只告诉家长或负责照顾你的家庭成员，不要发到群聊或公共截图中。

## 2. 孩子端导航

| 导航 | 可以做什么 |
| --- | --- |
| 今天 | 查看今天的任务和提醒 |
| 错题 | 复习还没有掌握的错题 |
| 成就 | 查看星星、家庭奖励和已完成的成长 |
| 我的 | 查看个人资料和退出孩子端 |

## 3. 查看今日任务

打开“今天”，在“今天的任务”中查看家长安排的内容。可以打开“待完成”的任务
查看详情；已经提交或已经确认的任务会显示当前结果。

常见状态：

- “待完成”：现在可以打开并提交完成情况。
- “等待家长确认”：已经提交，等家长检查。
- “已获得星星”：家长已经确认完成。
- “已取消”或“已归档”：任务不再需要提交。

“今日提醒”可能提示任务到期、错题复习或其他家庭提醒。如果某一类提醒暂时无法
加载，其余提醒仍然有效，可以稍后重新加载。

## 4. 提交任务完成情况

1. 在“今天”打开一个“待完成”任务。
2. 填写实际用时和实际完成数量。
3. 选择完成难度：轻松、刚刚好或有挑战。
4. 需要家长协助时，勾选“我需要帮助”。
5. 在“我的一句话”中写下感受或问题。
6. 选择“提交完成情况”。

成功后会显示“已提交，等待家长确认。”家长确认后，任务会进入完成状态并发放
星星。如果页面提示任务状态已经变化，返回“今天”重新打开最新内容。

## 5. 复习错题

打开“错题”可以看到还没有掌握的智育错题。孩子端只显示复习需要的信息和自己
填写的解释，不显示家长的私有备注。

1. 找到要复习的错题。
2. 在“我的解释”中写下自己的理解。
3. 还不确定时选择“我还不会”。
4. 已经理解并能正确完成时选择“我已经会了”。

选择“我已经会了”后，这道题会从待复习列表中移出。如果仍有疑问，应选择
“我还不会”并请家长帮助。

## 6. 查看成就和奖励

打开“成就”可以查看：

- “我的星星”
- “家庭奖励”
- “星星记录”
- “已经完成的成长”

达到奖励所需星星后，页面会提示“可以请家长兑换”。孩子端不能直接兑换或创建
奖励，请与家长确认后由家长在“星星与奖励”中操作。

## 7. 我的资料、退出和会话过期

打开“我的”可以查看姓名、年级、兴趣、需要加强的学科、喜欢的运动、艺术兴趣、
劳动习惯和品德目标。当前孩子端只供查看，不能直接修改资料。

在共享设备上使用完毕后，选择“退出孩子端”。如果会话过期，系统会返回登录
流程并提示重新登录；如果家长刚刚重设 PIN，必须使用新 PIN。

遇到暂时无法加载时，先选择页面上的重新加载操作。仍然失败时，告诉家长当前
页面和提示内容，但不要把 PIN 放进截图。
